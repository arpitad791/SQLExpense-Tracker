
# SQL Expense Tracker Database

This project contains a complete SQL-based solution for an expense tracking system.

## Contents
- **expense_tracker_schema.sql** – database tables and relationships
- **sample_data.sql** – sample data to test the system
- **analytical_queries.sql** – useful SQL queries such as:
  - total spending
  - spending by category
  - monthly trend
  - highest spending category

## How to Use
1. Create a new database in MySQL.
2. Run `expense_tracker_schema.sql` to create tables.
3. Run `sample_data.sql` to insert sample data.
4. Use `analytical_queries.sql` for analysis or demonstration.

This can be used as a portfolio-ready SQL project.
